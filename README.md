# Verilog-Digital-Designs
Design and simulation of digital circuits using Verilog HDL, covering combinational and sequential logic modules such as adders, multiplexers, demultiplexers, and flip-flops, implemented using dataflow and behavioral modeling techniques.

## 📌 About

Design and simulation of digital circuits using Verilog HDL, covering combinational and sequential logic modules such as adders, multiplexers, demultiplexers, and flip-flops, implemented using dataflow and behavioral modeling techniques.

---

## 🧩 Resources Included

### ➕ Half Adder
Design and implementation of a half adder using both dataflow and behavioral modeling to perform basic binary addition.

---

### ➕ Full Adder
Implementation of a full adder circuit for binary addition with carry input and output using Verilog HDL.

---

### 🔀 Multiplexer (MUX)
Design of multiplexers (e.g., 2x1, 4x1) to select one input from multiple inputs based on selection lines.

---

### 🔁 Demultiplexer (DEMUX)
Implementation of demultiplexers to route a single input signal to multiple outputs.

---

### 🔄 Flip-Flops
Design and simulation of sequential circuits such as SR, JK, D, and T flip-flops for memory and storage applications.

---

## 🛠 Tools Used
Xilinx ISE design suite
